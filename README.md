# \# CGI Employee Scheduling System

# 

# A web-based scheduling application designed for CGI employees to view and manage shifts across various access levels. This project modernizes the existing scheduling process by converting legacy Excel-based tracking into a dynamic, scalable web application.

# 

# \## What the Project Does

# 

# The CGI Employee Scheduling System provides a centralized platform for workforce management. It allows users with different permission levels—from general employees to administrators—to view up-to-date schedules in real-time. By moving away from static Excel sheets, the system ensures data integrity, improves accessibility, and streamlines the communication of shift changes.

# 

# \## Why the Project is Useful

# 

# \* \*\*Efficiency\*\*: Eliminates the manual overhead of updating and distributing Excel files.

# \* \*\*Accessibility\*\*: Employees can view their schedules from any web browser without needing specific spreadsheet software.

# \* \*\*Role-Based Access\*\*: Ensures that sensitive scheduling data is only visible or editable by authorized personnel.

# \* \*\*Centralized Data\*\*: Maintains a single source of truth for all staffing requirements, reducing scheduling conflicts.

#
